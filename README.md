# ElitePrime Farms

## Setup Instructions

Follow these steps to set up the ElitePrime Farms project on your local machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nastyma/eliteprime-farms.git
   cd eliteprime-farms
   ```

2. **Install dependencies**:
   Make sure you have [Node.js](https://nodejs.org/) installed. Then run:
   ```bash
   npm install
   ```

3. **Start the server**:
   After installing the dependencies, start the server by running:
   ```bash
   npm start
   ```

4. **Open in your browser**:
   Go to `http://localhost:3000` in your web browser to see the application running.

5. **Run tests (optional)**:
   To run the test suite, execute:
   ```bash
   npm test
   ```

If you encounter any issues, please open an issue in the repository.
